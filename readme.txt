Creates a disclaimer popup on the page where the short code is used.

== Installation ==
1. Install the plugin
2. Activate the plugin
3. Create disclaimers from the \"Disclaimers\" menu item
4. Place the short code referencing the disclaimer you would like to use in a page.
5. Thats it!
